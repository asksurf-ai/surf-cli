# surf-cli (legacy)

The Go CLI has moved to **[asksurf-ai/surf-cli](https://github.com/asksurf-ai/surf-cli)**.

This repository now hosts only the JavaScript packages:

- [`@surf-ai/sdk`](./js/packages/sdk) — JS/TS SDK for the Surf data platform
- [`create-surf-app`](./js/packages/create-surf-app) — scaffolding tool for Surf apps

## Install the CLI

```sh
curl -fsSL https://downloads.asksurf.ai/cli/releases/install.sh | sh
```

CLI source, issues, and releases: https://github.com/asksurf-ai/surf-cli
